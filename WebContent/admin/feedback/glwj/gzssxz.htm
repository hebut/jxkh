河北工业大学本科毕业设计（论文）教学工作实施细则

（2009年2月 制订）

本细则依据《河北工业大学本科毕业设计（论文）教学工作业务规范》、《河北工业大学本科毕业设计（论文）教学工作管理规定》，对毕业设计（论文）教学环节相关的教学文件、教学成果文档、教学业务文档、教学管理文档的内容和格式提出了具体要求。

本细则是河北工业大学本科毕业设计（论文）教学工作的具体标准，及本科毕业设计（论文）教学信息系统设计的工作依据。

第一部分 教学文件内容和格式要求

一、河北工业大学本科毕业设计（论文）教学大纲

1.1 篇首栏目信息：课程编号、适用专业、教学周数，教学大纲制订负责人（系主任）、教学大纲审批负责人（主管院长）、教学大纲制订日期。

1.2 教学目的和意义：以本专业培养方案为依据，阐明毕业设计（论文）教学环节与培养目标的关系。

1.3 教学内容要求：以本专业培养培养目标为依据，阐明毕业设计（论文）选题范围和毕业设计（论文）成果目标的要求。

1.4 教学过程要求：以《河北工业大学本科毕业设计（论文）教学工作业务规范》为依据，结合本专业培养特点，阐明毕业设计（论文）教学进度和阶段教学成果要求。

1.5 编辑和打印：PDF文本、5号宋体、1.5倍行距、首行缩进2字符。A4纸双面打印。

二、河北工业大学本科毕业设计（论文）选题指南

2.1 篇首栏目信息：适用专业、指南制订负责人（系主任）、指南制订日期。

2.2 选题范围：阐明与本专业培养目标相一致的毕业设计和毕业论文类别。毕业设计应按设计对象细分子类，如产品或部件设计、装备或装置设计、工艺或管理流程设计、控制或应用电路设计、控制或信息系统设计、建筑结构设计、建筑模型设计等。毕业论文应按研究性质细分子类，如基础理论研究、应用基础研究、技术应用或推广方式和方法研究、实验研究、调研研究、案例研究等。

2.3 参考题目：结合本专业教学实践列示。

2.4 自拟题目：说明限定条件。

2.5 编辑和打印：PDF文本、5号宋体、1.5倍行距、首行缩进2字符。A4纸双面打印。

三、河北工业大学本科毕业设计（论文）任务书

3.1    篇首栏目信息：专业（方向）、学生学号、姓名、班级，指导教师号、姓名、职称，毕业设计（论文）题目、任务书下达日期。

3.2    内容要求：阐明与毕业设计（论文）题目相关、需要通过毕业设计解决、或通过毕业论文研究的主要问题。后面应列出建议学生在毕业设计（论文）前期研读的重要参考资料（书目、论文、手册、标准等）。

3.3    方法要求：阐明与毕业设计（论文）问题解决和研究相关的实验、设计、调研方法和技术路线。

3.4    过程要求：提出毕业设计（论文）的周工作进度、工作质量、阶段成果要求。

3.5    编辑和打印：PDF文本、5号宋体、1.5倍行距、首行缩进2字符。A4纸双面打印。

第二部分 教学成果文档内容和格式要求

四、河北工业大学本科毕业设计（论文）前期和中期报告

4.1    篇首栏目信息：专业（方向），学生学号、姓名、班级，指导教师号、姓名、职称，毕业设计（论文）题目、报告提交日期。

4.2    前期报告内容要求：①毕业设计前期报告要求阐述工作过程，遇到的问题、解决问题的方法、效果、启示，任务书要求进度完成情况；②毕业论文的前期报告要求提交文献综述。

4.3    中期报告内容要求：①毕业设计中期报告要求提交阶段设计成果或实验结果；②毕业论文中期报告要求提交实地（现场）调研研究报告。

4.4    篇幅要求：5000字左右。

4.5    编辑和打印格式：WORD文本，正文页眉 “河北工业大学本科毕业设计（论文）前期（中期）报告”，正文五号宋体。A4纸单面打印。

五、河北工业大学本科毕业设计图纸

5.1 图框信息：参考专业设计标准，各专业自行规定。

5.2 图纸格式：参考专业设计标准，各专业自行规定。

5.3 打印格式：参考专业设计标准，各专业自行规定。

六、河北工业大学本科毕业设计说明书

6.1   篇首栏目信息：毕业设计（论文）题目、专业（方向），（学生学号）、姓名、班级，指导教师（教教工号）、姓名、职称。

6.2    内容排序：设计依据、参考标准、参数（配方）选择、计算方法、计算工具等。

6.3    编辑和打印格式：PDF文本（终稿）、WORD文本（初稿）、5号宋体、1.5倍行距、首行缩进2字符。A4纸双面打印。

七、河北工业大学本科毕业论文

7.1    篇首栏目信息：毕业设计（论文）题目、专业（方向），（学生学号）、姓名、班级，指导教师（教教工号）、姓名、职称。

7.2    内容排序：中文摘要、英文摘要、章节目录、正文、参考文献、致谢、附录

7.3    正文章目：引言（研究背景和研究内容）、文献综述（相关研究现状）、研究（资料数据收集）过程和方法、研究结果（资料数据处理）和讨论（分析）、结束语。

7.4    篇幅要求：正文1万字左右

7.5    编辑和打印格式：WORD文本（初稿）、PDF文本（终稿）、5号宋体、1.5倍行距、首行缩进2字符。A4纸双面打印。

第三部分 教学业务文档内容要求

八、河北工业大学本科毕业设计（论文）指导教师评语表

8.1 表头栏目信息：专业（方向）、学生姓名、学号、班级，指导教师姓名、职称、教教工号，毕业设计（论文）题目、成绩评定（平时、成果）。

8.2    工作质量评价：（提示：针对学生对待毕业设计（论文）工作态度认真程度、完成任务书规定的工作量和深度，实验、设计、调研过程严谨程度，设计说明书（论文）格式规范程度、文笔流畅程度等）。

8.3    成果质量评价：（提示：针对选题的意义；文献综述的规范性；实验、设计、调研过程的严谨性和方法的科学性；结论的逻辑性；成果的创新性）。

8.4    编辑打印格式：5号宋体，A4纸单面打印。

九、河北工业大学本科毕业设计（论文）同行评阅意见表

9.1 表头栏目信息：专业（方向）、学生姓名、学号、班级，评阅人姓名、职称、教工号，毕业设计（论文）题目、成绩评定（成果）。

9.2 成果质量评价：（提示：针对选题的意义；文献综述的规范性；实验、设计、调研过程的严谨性和方法的科学性；结论的逻辑性；成果的创新性）

9.3 编辑打印格式：5号宋体，A4纸单面打印。

十、河北工业大学本科毕业设计（论文）答辩记录和决议表

10.1  表头栏目信息：专业（方向）、学生姓名、学号、班级，答辩教师姓名、职称、教工号，毕业设计（论文）题目、成绩评定（成果、答辩）

10.2  答辩过程记录：（提示：据原始记录整理、归纳3-5个主要问题和主要回答录入）

10.3  答辩决议（成果评价）：（提示：针对选题的意义；文献综述的规范性；实验、设计、调研过程的严谨性和方法的科学性；结论的逻辑性；成果的创新性）

10.4  答辩决议（答辩表现）：（提示：陈述问题的清晰度、回答问题的正确度）

10.5  编辑打印格式：5号宋体，A4纸双面打印。

十一、河北工业大学本科优秀毕业设计（论文）推荐审批表

11.1   表头栏目信息：专业（方向）、学生姓名、学号、班级，推荐人姓名、职称、教工号，毕业设计（论文）题目、成果成绩（导师评分、评阅人评分、答辩小组评分）

11.2   推荐意见：从选题的意义；文献综述的规范性；实验、设计、调研过程的严谨性和方法的科学性；结论的逻辑性；成果的创新性评价，在上述五方面的评价尺度中，至少有四方面在本专业同届毕业设计（论文）成果中属于相对突出。

11.3   审核意见：答辩分委会主席（系主任）、答辩委员会主席（主管院长）分别签署，说明相对优秀的事实。

11.4   编辑打印格式：5号宋体，A4纸双面打印。

十二、河北工业大学本科毕业设计（论文）延期答辩登记表

12.1    表头栏目信息：专业（方向）、学生姓名、学号、班级，指导教师姓名、职称、教工号，毕业设计（论文）题目、成绩评定（平时、成果）。

12.2   延期答辩原因：成绩不及格（平时成绩、成果成绩）。

12.3   延期审核依据：系主任签署，附指导教师和评阅人评语。

12.4   编辑打印格式：5号宋体，A4纸单面打印。
